# Seestar S50

**Summary**: The ZWO Seestar S50 is a reference all-in-one smart telescope used as a benchmark for defining smart telescope traits.

**Sources**: SmartTelescope.md

**Last updated**: 2026-04-19

---

The Seestar S50 by ZWO is one of the two primary reference products in the [[smart-telescope]] category. ZWO positions it as an all-in-one integrated unit with automated GoTo, tracking, and [[plate-solving]] built in. (source: SmartTelescope.md)

## Key characteristics

- All-in-one form factor: optics, camera, mount, and computer in one unit
- Automated GoTo and star tracking
- Plate-solve-based alignment — no manual star alignment needed
- App-controlled: target selection, live preview, and sharing via smartphone
- [[live-stacking]] for deep-sky imaging
- Consumer-oriented: minimal setup time

## Role in this project

The Seestar S50 is used as a benchmark to define what features a [[hardware-platform]] build (C8 + Pi 5 + OnStep) must replicate to earn the "smart telescope" label. See [[requirements]] for the derived requirement set.
