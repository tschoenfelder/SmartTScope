# Vaonis Vespera Pro

**Summary**: The Vaonis Vespera Pro is a reference robotic telescope system used as a benchmark for smart telescope traits, alongside the Seestar S50.

**Sources**: SmartTelescope.md

**Last updated**: 2026-04-19

---

The Vaonis Vespera Pro is one of the two primary reference products used to define the [[smart-telescope]] category. Vaonis frames it explicitly as a "robotic telescope system" rather than a traditional OTA, emphasizing the autonomous and computational nature of the device. (source: SmartTelescope.md)

## Key characteristics

- All-in-one robotic telescope — not a traditional OTA with add-ons
- Autonomous alignment workflow via phone-provided time/location + astrometric alignment; no manual star alignment
- Automated slew, center, and track
- Camera-first: live digital imaging rather than eyepiece observing
- App-centered UX for target selection, session management, and sharing
- Advanced variant: supports multi-night capture, mosaic mode, environmental automation

## Role in this project

Alongside [[seestar-s50]], Vespera Pro defines the category ceiling. The more advanced features (mosaic, scheduling, multi-night) inform the "Full" tier of the [[requirements]] for the C8 + Pi 5 + OnStep build.
