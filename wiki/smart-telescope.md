# Smart Telescope

**Summary**: Defines what distinguishes a "real" smart telescope from a traditional telescope with app control, and identifies the core traits that define the category.

**Sources**: SmartTelescope.md

**Last updated**: 2026-04-19

---

A smart telescope is more than a telescope with an app. It behaves as a **self-contained robotic observing system** that a beginner can set up, point, and get results from without classical astrophotography knowledge.

## Defining traits

The canonical formula is:

> **portable + integrated + self-aligning + self-pointing + self-tracking + computationally imaging + app-controlled**

A device that meets all seven traits is generally accepted as a smart telescope. A traditional telescope with a GoTo mount or a camera attachment does not qualify. (source: SmartTelescope.md)

### All-in-one integration
Optics, camera, mount, focuser, onboard computer, and practical extras (dew heater, filters) are built into one unit. No laptop, guide setup, or separate capture chain is required. (source: SmartTelescope.md)

### Autonomous setup and alignment
The device determines its own orientation automatically — typically via phone-provided time/location plus [[plate-solving]] — so the user never performs classical star alignment. (source: SmartTelescope.md)

### Automated target acquisition and tracking
The user selects an object in the app; the telescope slews, centers it, and keeps tracking with minimal intervention. The "tap target → observe" behavior is one of the strongest markers of the category. (source: SmartTelescope.md)

### Computational imaging
Smart telescopes are camera-first, not eyepiece-centric. They capture, stack, enhance, and display a live improved image on a phone or tablet. See [[live-stacking]]. (source: SmartTelescope.md)

### App-driven UX
The app is the main interface for everything: target selection, session monitoring, mode adjustment, firmware updates, and result sharing. (source: SmartTelescope.md)

### Consumer-grade ease of use
Works for beginners in minutes. Low skill barrier is a defining category trait, not a marketing extra. (source: SmartTelescope.md)

## Basic vs. advanced smart telescopes

Features that separate a basic smart telescope from a more advanced one:

- Multi-night capture, scheduling, robotic unattended operation
- Mosaic modes for large targets
- Better onboard and post-processing
- Higher-resolution sensors and more ambitious imaging modes
- Environmental automation (anti-fog, longer battery endurance)

(source: SmartTelescope.md)
